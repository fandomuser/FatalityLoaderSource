# Технический Стек (Technology Stack)

## Какой GUI используется?
Этот проект написан на **WPF (Windows Presentation Foundation)** с использованием **.NET 8**.

### Почему WPF, а не WinAPI?
*   **WinAPI**: Это старая технология (на C++), которая используется для создания максимально легких приложений. На ней очень сложно делать красивый дизайн (тени, градиенты, прозрачность). Код на WinAPI для такого интерфейса занял бы в 10 раз больше времени.
*   **WPF**: Это современная технология (на C# + XAML). Она позволяет легко создавать сложные интерфейсы, как у Fatality (с тенями, закруглениями, картинками).

## Почему размер файла 155 МБ?
Это цена за удобство и "независимость".

1.  **Self-Contained (Автономный)**: Мы собрали программу так, чтобы она работала **везде**, даже если у пользователя нет .NET. Для этого пришлось "засунуть" внутрь `.exe` файла весь .NET Runtime (движок, библиотеки, сборщик мусора и т.д.). Это занимает около 150 МБ.
2.  **Framework-Dependent (Зависимый)**: Если бы мы не включали Runtime, файл весил бы всего **~200 КБ**. Но тогда вашему другу пришлось бы самому скачивать и устанавливать **.NET Desktop Runtime 8.0**, иначе программа бы не запустилась.

**Итог:**
*   **WPF** дает красивый дизайн.
*   **Self-Contained** дает запуск без установки драйверов/библиотек.
*   **155 МБ** — это вес всего этого удобства.
